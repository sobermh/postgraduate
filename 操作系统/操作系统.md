# 常见工具使用与性能分析
## zsh

（强大的shell解释器）
apt：软件管家，安装工具
```
sudo apt update       #更新软件列表
sudo apt upgrade      #升级软件列表
sudo apt install zsh  #下载安装zsh
chsh -s /bin/zsh      #shell默认解释器修改为zsh
```
- zsh配置文件
etc:存储配置文件（全局）
```
~/.zshrc           /etc/zshrc
~/.zlogin          /etc/zlogin
~/.zlogout         /etc/zlogout
~/.zprofile        /etc/zprofile
```
- zsh命令行操作
ctrl+a 回到行首
ctrl+e 回到行尾
